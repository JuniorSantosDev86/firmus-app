# Firmus App

Web-first MVP foundation for Firmus, a modern SaaS for service providers and solo operators.

## Tech base

- Next.js (App Router)
- TypeScript
- Tailwind CSS v4
- shadcn/ui baseline

## Local commands

```bash
npm run dev
npm run lint
npm run build
npm run start
```

## Current status

This repository is in the foundation stage and intentionally keeps the app shell minimal.
